=== Modern Notepad Wordpress Theme ===
Tags: two-columns, fixed-layout, white, custom-header

= Description =
A very stylish modern notepad skin, designed for people who see their blogs as a modern notepad and want it to look alike.
*Note: This version supports only one level of navigation menus. Only a single row of buttons will be displayed in the main navigation area, regardless of how many items are in your navigation menu. The total number of items visible depends on the width of the navigation button text.

== Installation ==

= Via WordPress Admin =
1. From your sites admin, go to Themes > Install Themes
2. In the search box, type 'Modern Notepad' and press enter
3. Locate the entry for 'Modern Notepad' and click the 'Install' link
4. When installation is finished, click the 'Activate' link

= Manual Install =
1. Unzip this file into your mywebsite.com/wp-content/themes/ directory
2. Go into your Wordpress Admin, navigate to 'Appearance > Themes'
3. Find the 'Modern Notepad' listing on this page and click 'Activate'
