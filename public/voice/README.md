# ミリネ教科書 音声ファイル

このフォルダに以下の音声ファイルを配置してください。

## 必要なファイル構成

```
voice/
├── 1kyu/
│   ├── Mi1-Main1.mp3 ～ Mi1-Main25.mp3  (本文)
│   └── Mi1-sub1.mp3 ～ Mi1-sub25.mp3    (文法・語彙)
├── 2kyu/
│   ├── Mi2-Main1.mp3 ～ Mi2-Main24.mp3  (本文)
│   ├── Mi2-Bunpo1.mp3 ～ Mi2-Bunpo24.mp3 (文法)
│   └── Mi2-Goi1.mp3 ～ Mi2-Goi24.mp3    (語彙)
├── 3kyu/
│   ├── Mi3-Main1.mp3 ～ Mi3-Main24.mp3  (本文)
│   └── Mi3-Sub1.mp3 ～ Mi3-Sub24.mp3    (文法・語彙)
└── 4kyu/
    ├── Mi4-Main1.mp3 ～ Mi4-Main24.mp3  (本文)
    └── Mi4-Sub1.mp3 ～ Mi4-Sub24.mp3    (文法・語彙)
```

ファイルを配置後、デプロイすると https://mirinae.jp/voice/ で音声が再生されます。
